## Docker-LNMP

### Introduction

From [PHPDocker.io](https://phpdocker.io/) generated.

---

### install
```bash
docker-compose up -d 
```

### init
```bash
mkdir nginx
touch nginx/nginx.conf
```

---

### Usage

**nginx**  
setting server in `./nginx/nginx.conf`

**php**  
setting php.ini in `./php73/php-ini-overrides.ini`

**phpmyadmin**  
[localhost:8080](localhost:8080)
